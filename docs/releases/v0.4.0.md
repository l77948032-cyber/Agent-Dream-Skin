# DreamSkin Studio v0.4.0

产品版本 `0.4.0` 是 DreamSkin 从内置 Agent 连接模式转向“本地主题库 + 通用 CLI”的首个公开
测试版，同时修复了早期 `0.3.0` 构建的同版本运行时冲突。

## 本版亮点

- Studio 现在专注于本地主题管理与完整 UI 预览，像桌面端 Obsidian 一样管理“我的主题”。
- Studio 不连接 Agent，也不提供内置 Agent 对话或额外服务进程。
- 主题中心扩充为 20 套可添加模板：Trae 10 套、WorkBuddy 10 套，并保留空白主题入口。
- 新增随桌面应用交付的 `dreamskin` CLI。用户正在使用的任意编程 Agent 都可通过
  `targets` 和 `theme inspect|list|read|create|update|asset import|validate` 操作同一套本地主题。
- CLI 修改会被 Studio 自动发现并刷新预览。
- CLI 不提供应用、恢复或删除命令；这些高影响操作仍需用户在 Studio 中明确触发。

## 升级修复

- 将 DreamSkin Studio、Trae 运行时与 WorkBuddy 运行时统一升级到 `0.4.0`。
- 从早期 `0.3.0` 构建升级时，新运行时安装到独立版本目录，不再因内容不同而触发
  `RUNTIME_VERSION_CONFLICT`。
- 原有 `0.3.0` 运行时会保留为可回退版本，不覆盖或删除旧版本内容。
- 已有主题、主题库、预览图、备份和目标应用状态都会保留。

## 下载与安装

本次公开未签名测试版使用 tag `test-v0.4.0-macos-arm64`：

- 推荐下载：[`DreamSkin-Studio-0.4.0-mac-arm64.dmg`](https://github.com/l77948032-cyber/Agent-Dream-Skin/releases/download/test-v0.4.0-macos-arm64/DreamSkin-Studio-0.4.0-mac-arm64.dmg)
- 备用下载：[`DreamSkin-Studio-0.4.0-mac-arm64.zip`](https://github.com/l77948032-cyber/Agent-Dream-Skin/releases/download/test-v0.4.0-macos-arm64/DreamSkin-Studio-0.4.0-mac-arm64.zip)
- 文件校验：[`SHA256SUMS.txt`](https://github.com/l77948032-cyber/Agent-Dream-Skin/releases/download/test-v0.4.0-macos-arm64/SHA256SUMS.txt)

下载后先运行下面的命令，并确认结果与 `SHA256SUMS.txt` 中 DMG 对应的一行完全相同：

```bash
shasum -a 256 ~/Downloads/DreamSkin-Studio-0.4.0-mac-arm64.dmg
```

打开 DMG，把 **DreamSkin Studio** 拖到 **Applications**。ZIP 仅作为备用下载格式。

此测试构建使用 ad-hoc 签名，尚未完成 Apple Developer ID 公证。其他用户可以公开下载，
但 macOS 可能显示开发者验证提示；应用内自动更新不会在此构建中启用。

首次启动若提示无法验证开发者，请在确认 SHA-256 匹配后，到“应用程序”中按住 Control 点击
（或右键点击）**DreamSkin Studio**，选择“打开”，然后在确认框中再次选择“打开”。如果仍被
阻止，可前往“系统设置 -> 隐私与安全性”，在安全性提示旁选择“仍要打开”。校验不匹配时不要
绕过系统保护，请删除文件并从本项目 Release 重新下载。

## 兼容性

- macOS 12 或更高版本。
- Apple silicon（arm64）。
- Trae 当前实机适配基线：TRAE SOLO CN `0.1.36`。
- WorkBuddy 当前实机适配基线：`5.2.6`。
- 使用桌面安装包内的 `dreamskin` CLI 不要求另外安装 Node.js。
