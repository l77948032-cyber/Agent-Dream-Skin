# DreamSkin Studio v0.4.1

产品版本 `0.4.1` 完成了 Trae 国际版与 TRAE SOLO CN 的双版本适配，并让 Studio 能识别和显示
当前实际宿主与主题运行状态。

## 本版亮点

- 同一套 Trae 主题现在可以应用到 Trae 国际版或 TRAE SOLO CN。
- Studio 会识别当前宿主，显示国际版/CN 版名称、版本和主题运行状态。
- 国际版使用标准 Workbench 页面，CN 版保留 Work、Code、Design 与对话页展示；主题会按实际
  页面自动选择对应效果，不要求用户维护两套主题。
- 两个版本同时安装时，DreamSkin 优先跟随已经打开的 Trae；如果两个版本同时运行，会停止
  操作并要求用户明确保留一个，避免把主题应用到错误应用。
- 应用、验证、切换和恢复始终绑定到同一个 Trae 宿主，不会在国际版与 CN 版之间串用状态。

## 升级说明

- DreamSkin Studio、Trae 运行时与 WorkBuddy 运行时统一升级到 `0.4.1`。
- `0.4.1` 会安装到新的运行时版本目录，已有 `0.4.0` 不会被覆盖；已有主题和预览仍会保留。
- 从旧版升级后，首次应用主题会自动准备新运行时，无需手工清理应用数据。

## 兼容性与验证范围

- macOS 12 或更高版本，Apple silicon（arm64）。
- TRAE SOLO CN：当前本机官方应用版本 `0.1.38`，身份与签名检查通过。
- Trae 国际版：按官方 `3.5.78` 应用包完成身份、签名和标准 Workbench 自动化兼容测试；本次
  未在本机安装国际版进行人工启动验收。
- WorkBuddy：当前实机适配基线 `5.2.6`。
- 使用桌面安装包内的 `dreamskin` CLI 不要求另外安装 Node.js。
